# Weather Forecast API - RESTful Weather Service

## Overview 
RESTful API service providing weather forecast data with caching and rate limiting. 
 
## Features 
- RESTful API endpoints 
- Weather data aggregation 
- Response caching 
- Rate limiting 
- API documentation 
 
## Tech Stack 
- **Backend:** Python, Flask 
- **Cache:** Redis 
- **API:** OpenWeatherMap
